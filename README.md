# Python学生管理系统 python105

## 项目概述

本项目是基于Python的学生管理系统，旨在提供一套完整的教务管理系统，涵盖学生、课程、成绩等多方面管理功能。

## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：Vue.js

## 功能模块

### 基础功能

- 登录注册

### 管理功能

- 学生管理
- 课程管理
- 学生成绩管理
- 选课信息管理
- 课程类型管理
- 公告管理
- 教师管理
- 个人中心

## 系统角色

- 学生
- 教师
- 管理员

## 运行环境

- Python环境
- MySQL数据库

## 部署步骤

1. 配置Python环境
2. 安装所需依赖
3. 配置MySQL数据库
4. 运行系统
